# Temperature Data Analysis

## Description
This project analyzes daily temperature data using Python.  
It calculates weekly and monthly average temperatures and visualizes them using line plots.

## Files
- Data.csv – Input temperature data  
- temperature.py – Python script  
- README.md – Documentation  

## Requirements
- pandas  
- numpy  
- matplotlib  

## Data Format
The CSV file should contain:
- Date
- Temperature

## Output
- Daily temperature variation graph  
- Weekly average temperature graph  
- Monthly average temperature graph  

## How to Run
python Temperature_Analyzer.py
